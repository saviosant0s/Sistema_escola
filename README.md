# Sistema de Gerenciamento Escolar (Terminal)

Projeto em Python desenvolvido para praticar lógica de programação e Programação Orientada a Objetos (POO), simulando o funcionamento básico de um sistema de secretaria escolar.

---

## Sobre o projeto

O sistema permite cadastrar, editar, excluir e visualizar alunos por meio de um menu interativo no terminal.  
As turmas são organizadas automaticamente e todos os dados são armazenados apenas em memória, sem uso de banco de dados.

Este projeto foi desenvolvido com foco em aprendizado, organização de código e prática de conceitos fundamentais da linguagem Python.

---

## Funcionalidades

- Cadastro de alunos com matrícula automática  
- Distribuição automática entre turmas (132 e 133)  
- Listagem de alunos por turma  
- Relatório geral em ordem alfabética  
- Edição de dados e transferência entre turmas  
- Exclusão de alunos  
- Dados fictícios para testes  

---

## Tecnologias e conceitos

- Python 3  
- Programação Orientada a Objetos (POO)  
- Listas e dicionários  
- Manipulação de strings  
- Organização e reutilização de código
